# Sugui_HanahLorein_ShoppingCartActivity
This project was developed as part of a learning activity in C# programming, with the assistance of AI tools used as a study aid. The AI was used to help clarify programming concepts and to support my understanding of how different parts of the system work together, especially in object-oriented programming, input handling, and program logic design.

Instead of directly copying solutions, I used AI responses as references to guide my thinking and improve my implementation. This helped me understand how to structure classes, manage data in arrays, and apply validation to user inputs within a console-based application.

Throughout the development process, I encountered challenges such as organizing the program flow, ensuring accurate calculations for the shopping cart total, and improving the clarity of the console interface. I addressed these issues by revising my code step by step and testing each feature as it was added.
